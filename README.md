# 來動動

這是一個可以加入手機桌面的純前端 Web App。它會在瀏覽器本機記錄每天早上、晚上是否完成，不需要後端。

## 使用方式

1. 開啟 `index.html`。
2. 點右上角設定，填入早上影片網址和晚上影片網址。
3. 每天點「早上」或「晚上」會打開對應影片，並自動記錄完成。
4. 回顧區可以切換最近 7 天或 30 天，直接點集點章上的「早」或「晚」補記。
5. 連續 3 天早晚都完成時，畫面會噴彩帶慶祝。

## 加到手機桌面

建議把這個資料夾部署到 GitHub Pages、Netlify、Vercel，或任何 HTTPS 靜態網站服務，再用手機瀏覽器打開網址並選擇「加入主畫面」。

如果只是在電腦本機測試，也可以用本機預覽伺服器打開；但真正放到手機桌面時，HTTPS 網址最穩。
